for...of

for-of is a new loop in ES6 that replaces both for-in and forEach() and supports the new iteration protocol.

Syntax

for (variable of iterable) { statement }

Iterating over an Array

let values = [10, 20, 30]; for (let value of values) { console.log(value); } // 10 // 20

Iterating over a String

let color = "red"; for (let c of color) { console.log(c); } // r // e // d

Difference between for...of and for...in
